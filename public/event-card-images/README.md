# Event Card Images

Картинки для карточек событий «Собрались».

Основная папка для универсальных изображений:

- `common` — общие картинки, которые подходят под разные мероприятия.

Если позже захочется разделять изображения по темам, можно использовать тематические папки:

- `breakfast` — завтраки;
- `bath` — бани и банные вечера;
- `dinner` — ужины и камерные встречи;
- `club` — клубные события, мастермайнды, женские круги;
- `other` — остальные сценарии.

Подходящие форматы: `.jpg`, `.jpeg`, `.png`, `.webp`.

Лучше использовать горизонтальные или квадратные изображения хорошего качества.
