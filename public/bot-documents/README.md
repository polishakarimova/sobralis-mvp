# Документы для Telegram-бота «Собрались»

Эта папка предназначена для документов, которые открываются из Telegram-бота на первом запуске.

Сейчас бот ссылается на:

- `privacy-policy.html` — политика конфиденциальности
- `personal-data-consent.html` — согласие на обработку персональных данных
- `marketing-consent.html` — согласие на рекламную рассылку

Если позже документы будут в PDF, можно либо заменить ссылки в `lib/telegram.ts`, либо положить рядом PDF-файлы и обновить кнопки бота на новые имена.
