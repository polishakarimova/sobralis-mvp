Картинки для карточек событий лежат по темам:

- `breakfast` — завтраки
- `bath` — банные вечера
- `lunch` — бизнес-ланчи
- `other` — другое

Как добавлять:
1. Откройте нужную папку темы.
2. Положите туда горизонтальные изображения.
3. Проще всего заменять файлы с такими же именами:
   - `1.svg`
   - `2.svg`

Можно использовать `jpg`, `png`, `webp` или `svg`. Если имя или расширение будет другим, нужно поменять путь в `app/page.tsx` в массиве `eventVisuals`.

Рекомендуемый формат: горизонтальная картинка 1200x800 или 1600x1000.
