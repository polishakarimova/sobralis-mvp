# Sobralis Brand Guide

## Суть бренда

«Собрались» — стильный digital-сервис для организации красивых групповых встреч.

Сценарии:

- завтраки;
- банные встречи;
- клубные мероприятия;
- женские круги;
- мастермайнды;
- камерные встречи;
- выезды;
- небольшие платные события.

Для гостя сервис выглядит как красивая карточка приглашения.  
Для организатора сервис работает как умный помощник: гости, места, оплаты, статусы, ожидание, напоминания.

## Главная идея

Не “CRM для мероприятий”, а:

> красивая карточка приглашения + тихий организаторский слой.

## Визуальный характер

- эстетичный;
- спокойный;
- премиальный;
- тёплый;
- женственный, но не сладкий;
- технологичный, но не холодный;
- красивый, но не декоративный ради декора.

## Главный образ

Карточка приглашения — главный герой интерфейса.

Организаторские функции должны быть рядом или вторым слоем:

- места;
- оплаты;
- состав;
- ожидание;
- напоминания;
- ссылка-приглашение.

## Логотип

Название лучше писать:

`Собрались`

Допустимая signature-версия:

`Собрались.`

Логотип должен быть:

- мягкий;
- элегантный;
- читаемый;
- не слишком жирный;
- не слишком fashion;
- не свадебный;
- не wellness-only.

## Символ

Фирменный знак:

- центральная точка = событие;
- 4 точки вокруг = люди / участники;
- тонкие дуги = сбор вокруг события;
- общий смысл = «точка сбора».

Знак не должен быть:

- цветком;
- солнцем;
- SPA-символом;
- религиозным знаком;
- техническим радаром.

## Палитра

Основные цвета:

- Молочный фон: `#F5EFE6`
- Поверхность: `#FFFDF8`
- Шалфей: `#7E8466`
- Глина: `#B88768`
- Карамель: `#B78346`
- Тёплое золото: `#C59A55`
- Графит: `#2B2A27`
- Мягкий текст: `#7C746A`
- Border: `rgba(43,42,39,0.12)`

Использовать аккуратно:

- шалфей — для CTA и статуса;
- карамель/золото — для акцентов;
- графит — для текста;
- молочный — для фонов;
- розовый/глина — только мягко, не как главный цвет.

## Типографика

Display / brand / invitation:

- elegant serif;
- крупные заголовки;
- много воздуха;
- без чрезмерного жирного начертания.

UI:

- clean sans;
- небольшие подписи;
- статусы;
- кнопки;
- чипы;
- метаданные.

## Язык интерфейса

Предпочтительные тексты:

- «Красиво собрать своих»
- «Гости, места, оплаты и напоминания — в одной ссылке»
- «Я иду»
- «Создать событие»
- «Событие собрано»
- «Ссылка скопирована»
- «Осталось 6 мест»
- «10 из 12 оплачено»
- «2 человека в ожидании»

Избегать:

- “CRM”
- “панель управления”
- “администрирование”
- “таблица”
- “система контроля”
