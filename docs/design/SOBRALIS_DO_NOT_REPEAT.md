# Sobralis Do Not Repeat

## Ошибки, которые нельзя повторять

### 1. Огромный hero-заголовок

Не делать на первом экране огромный текст, который забирает внимание у карточки приглашения.

Плохо:
“Красиво собрать своих” огромным жирным заголовком на весь экран.

Лучше:
логотип → маленький слоган → красивая карточка приглашения.

### 2. Логотип как обычный шрифт

Не делать wordmark просто жирным serif без характера.

Логотип должен быть:
- элегантный;
- читаемый;
- не тяжёлый;
- с хорошим кернингом;
- связан со знаком.

### 3. Символ как SPA/солнце

Знак не должен выглядеть как:
- wellness;
- цветок;
- солнце;
- религиозный символ;
- орнамент.

Смысл знака:
- центральная точка = событие;
- 4 точки = люди вокруг;
- дуги = собрались вокруг события.

### 4. CRM на первом экране

Не показывать dashboard главным героем публичной части.

Главный герой:
- invitation card.

Второй слой:
- места;
- оплаты;
- состав;
- ожидание;
- напоминания.

### 5. Растянутая страница

Brand-preview не должен выглядеть как длинный лендинг.
Он должен ощущаться как компактный brand board.

### 6. Слишком декоративная ботаника

Не делать:
- flower shop;
- wedding stationery;
- много листьев;
- floral frames.

Ботаника должна быть:
- абстрактная;
- тонкая;
- почти фоновая;
- как намёк на композицию, а не тема бренда.

### 7. Дешёвый Canva-вид

Избегать:
- случайных иконок;
- слишком много карточек;
- разных радиусов;
- разных теней;
- плохих отступов;
- случайной типографики.
