# Sobralis Reference Analysis

## Главный визуальный референс

Файл:
`docs/design/references/sobralis-brand-reference.png`

Этот референс является источником визуальной правды для нового дизайна «Собрались».

Перед любыми задачами по:
— логотипу;
— brand-preview;
— публичной карточке события;
— UI-компонентам;
— палитре;
— типографике;
— invitation card;
— organizer layer;

нужно сверяться с этим изображением.

Главные элементы, которые нужно сохранять:
— мягкий молочный фон;
— премиальная карточка приглашения;
— крупный elegant serif для названия события;
— спокойный sans-serif для UI;
— фирменный знак “точка сбора”;
— карточки организатора вокруг главной карточки;
— палитра шалфей / глина / карамель / графит / молочный;
— мягкие тени;
— тонкие линии;
— ощущение ritual & rest / premium invitation UI;
— не CRM, а красивая ссылка-приглашение.

Что нельзя терять:
— компактность композиции;
— эстетичность;
— много воздуха;
— мягкую премиальность;
— ощущение “такую ссылку хочется отправить гостям”.

Что нельзя делать:
— не превращать в длинный лендинг;
— не делать огромные жирные заголовки;
— не делать CRM-dashboard главным героем;
— не делать Canva-шаблон;
— не делать SPA/цветочный/свадебный стиль;
— не придумывать новый визуальный стиль без сверки с референсом.

## Эталонное направление

Основной визуальный референс бренда «Собрались»:

Premium botanical social club / ritual & rest / elegant invitation UI.

Это не CRM, не дашборд, не обычный SaaS и не праздничная открытка.

Главная идея визуала:

> красивая карточка приглашения + спокойный организаторский слой.

## Что должно чувствоваться

- красиво;
- спокойно;
- дорого;
- эстетично;
- тепло;
- собрано;
- тактильно;
- современно;
- приглашение, в которое хочется попасть.

## Главный герой интерфейса

Не dashboard.
Не таблица.
Не список функций.

Главный герой — красивая карточка приглашения.

Организаторские функции должны быть вторым слоем:
- места;
- оплаты;
- состав;
- ожидание;
- напоминания;
- ссылка-приглашение.

## Что понравилось в референсе

- мягкий молочный фон;
- крупная карточка приглашения;
- elegant serif typography;
- спокойный sans-serif для UI;
- мягкие карточки вокруг основной карточки;
- тонкие линии-связи;
- ощущение premium brand board;
- логотип с символом “точка сбора”;
- палитра шалфей / глина / карамель / графит;
- UI не выглядит как CRM;
- карточка выглядит как приглашение, а не форма.

## Что нельзя терять

- ощущение красивого приглашения;
- много воздуха;
- мягкий светлый фон;
- тонкие линии;
- премиальную типографику;
- компактность брендборда;
- спокойную палитру;
- символ “люди вокруг события”.

## Что нужно улучшать относительно текущих черновиков

- не делать огромный hero-заголовок;
- не растягивать страницу как лендинг;
- не делать логотип грубым;
- не делать wordmark слишком жирным;
- не делать символ похожим на SPA/солнце/цветок;
- не превращать карточку в CRM;
- не перегружать нижний brand board;
- не делать много мелких блоков ради функций.

## Формула хорошего результата

Если пользователь открывает `/brand-preview`, он должен подумать:

“О, это красиво. Я бы хотела отправить такую ссылку гостям.”

А потом уже:

“И ещё тут удобно: места, гости, оплаты и напоминания собраны в одном месте.”
